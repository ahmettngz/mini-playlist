# Mini Playlist

Bu proje, derslerde öğrendiğim bilgileri pekiştirmek amacıyla oluşturulmuş küçük bir müzik çalar uygulamasıdır.

## Özellikler

- **Müzik Çalma**: Seçilen şarkıyı çalma ve duraklatma özelliği.
- **Ses Kontrolü**: Ses seviyesini ayarlama imkanı.
- **Şarkı Listesi**: Çalma listesinden şarkı seçebilme.

## Kurulum

Henüz bir kurulum adımı bulunmamaktadır.

## Kullanım

Uygulamayı başlatmak için:

```bash
python main.py
```

## Teknolojiler

- Python
- Tkinter (GUI)
